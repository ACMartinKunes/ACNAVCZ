Create new VAT Check Report on VAT Check Report Card. Fill field XML Schema on FastTab Other.

Suggest lines by action Fill report Lines.

The calculation of the items is built on the same logic as calculation for the VAT statement, i.e. user can specify whether are included entries Open, Closed or Open and Closed, Within Period or Before and Within Period. Entries with VAT date 31.12. 2013 and lower are not included. For simple check, you can display the totals of entries read in the particular sections of VAT Check Report with the possibility of displaying the documents from wich the displayed amount is consisted - by action Preview.

User modifications are available. You can use action Edit. After edition is finished, it is necessary to recalculate VAT Check Report entries - the system automatically asks the user whether the entries should be recalculated - the recalculation of entries is important especially when the VAT Check Report sections are changed.

Export from VAT Check Report Preview by action Launch export to XML.

Close VAT Check Report by action Close after data checking.
