Cash Desks for Till Documents

This code list is used to link the number series with a sales ticket office in different currencies. The number series are then established rights to data and default values documents, which allows you to the user much easier to work with cash register sales documents and client to execute much faster. The number serie should have set the default numbering, and the completed documents to the appropriate group for sales documents and cash cash register sales credit memos have given preset some values (see default values documents). At the same time the number serie must have a defined context and relationships.
