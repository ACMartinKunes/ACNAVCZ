List of entities for which occurred the reasons for the unregistering payer VAT is available on the website of financial management. The import will start using action Mass Uncertainty Payer get on page Uncertainty Payer Entries. Select import by legislation. The system checks the data on the cards of customers (Last Uncertainty Check Date, VAT Uncertainty Payer), Vendors and set field VAT Uncertainty Payer to Yes. In the case that the customer/vendor is not in the list, the indication Unreliable the payer is empty. Last Uncertainty Check Date is transferred from import file. When you create a new card Customer/Vendor, update entries by action Check All. When the system finds a match for VAT Registration No. in the table Registration County/Region then the system set field Uncertainty Payer to IN COUNTRY.
