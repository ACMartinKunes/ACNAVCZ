For the purposes of reporting the Slovak the legislation is used on the system NAV standard functionality of Account Schedules with adaptations that enable you to export each accounting statement on the portal, financial management, SR.

Individual lines of statements, balance sheet and profit and loss account, are defined in the module Departments / Financial Management / Reports and Analysis / Account Schedules. You can choose for which legislation is statement used in the field Assing Legislation . The field Assign Legislation is used to separation of SK and CZ statements through the filter.

To export the financial statement to the xml must have individual lines set up mapping in the auxiliary table according to statements provided for the financial management of the SR. Is needed to map each comparison period and row/column individually. This means that for each line of the statement will set the mapping of the current and the previous period.

On the export the account schedule is used the standard functionality of account schedules through the function keys Save Results and Results . Export Account Schedule from Acc. Schedule Res. Header List.

The results of the accounting schemes are stored in the chronologic sequence of their occurrence.

For Export you can use action Export do XML. Page appears in which you reside for more fields for export.

You can then import the saved file to the portal or to the application of the financial administration eDane .

From 31.12. 2014 you must import the balance sheet and income statements from one xml file. To link the file balance sheets and profit and loss account use option Export Merged Acc. schedule to XML. You must enter the result code for the balance sheet and the income statement and the previous period. Then just import the resulting file into an electronic form of financial management – UZPODv14 - ÚÈTOVNÁ ZÁVIERKA.
