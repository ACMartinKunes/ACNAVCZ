## manage alerts

Add-on module allows you to manage alerts system notification to States data and processes in Microsoft Dynamics NAV system. the alarm works both internally in the system environment, as well as externally in the form of an e-mail. The basis for the use of the module is to define the conditions and monitored events in the system, on which it is necessary to notify the user or respond in a particular way. In addition to monitoring current events you can monitor the development of events in time.
