# <a name="fp-pack-tracking-return-packing"></a>Evidence vratných obalů

Add-on modul Evidence vratných obalů je rozšířením modulu Zásoby systému Microsoft Dynamics NAV. Umožňuje sledování oběhu vratných obalů a paletového hospodářství. Lze pomocí něj sledovat obalová salda obchodních partnerů s vazbou na příslušné obchodní transakce (Nákup, Prodej, Příjem, Výdej, Transfer, Spotřeba a Výroba) a dále pak sledovat aktuální cenu obalů.
